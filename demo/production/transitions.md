# 🎞️ VoltAudit AI Transition Guidelines

Recommended transition styles between scenes:

- **Intro Card to Dashboard Home:** **Fade In (1.0s)**
- **Dashboard Home to Architecture Diagram:** **Cross Dissolve (0.5s)**
- **Architecture to AI Workforce Diagram:** **Cross Dissolve (0.5s)**
- **Diagrams to Demo Walkthrough:** **Zoom Cut (0.3s)**
- **Clean Run Demo to Dirty Run Demo:** **Whip Pan (0.2s)** (indicates change in state)
- **Dirty Run Demo to Human Override Queue:** **Cross Dissolve (0.5s)**
- **Override Queue to Swagger API docs:** **Cut**
- **Swagger Docs to Outro Card:** **Fade Out to Black (1.5s)**
