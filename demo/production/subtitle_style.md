# 🔤 VoltAudit AI Subtitle Style Guide

Style specifications for caption overlays:

- **Typography:**
  - Font Family: **Outfit** (or Inter fallback).
  - Font Weight: **Medium** (500).
  - Font Size: **24pt** (for a standard 1080p frame).
- **Colors:**
  - Text Color: **#FFFFFF** (Pure White).
  - Highlight Color: **#3B82F6** (Bright Blue for active words if using dynamic captions).
- **Background & Positioning:**
  - Placement: **Lower-third** (aligned center, 10% offset from bottom edge).
  - Bounding Box: Semi-transparent dark slate background block (**#0F172A** with **80% opacity**), with rounded corners (4px).
  - Padding: **8px top/bottom, 16px left/right**.
